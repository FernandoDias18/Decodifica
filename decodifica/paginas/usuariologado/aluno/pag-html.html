HTML

HTML(HyperText Markup Language) é uma linguagem de marcação utilizada na construção de páginas Web.

Neste curso você entenderá tudo sobre a linguagem HTML, aprendendo sobre a estrutura e funcionamento básico de uma Página Web. Ao finalizar este curso, você será capaz de estruturar uma página na internet e estará pronto para seguir seu caminho como desenvolvedor aprendiz no Decodifica :)
