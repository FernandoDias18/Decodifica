JavaScript

JavaScript(JS) é uma linguagem de programação com tipagem dinâmica fraca que, juntamente com HTML e CSS, é uma das três principais tecnologias utilizadas na Web. JavaScript permite páginas da Web interativas e, portanto, é uma parte essencial dos aplicativos.

Através do Curso JavaScript Decodifica você será capaz de se tornar um desenvolvedor completo, aprendendo os fundamentos básicos e intermediários da linguagem bem como a criação de páginas e projetos interativos para adicionar ao seu portfólio. Vem com a gente!)
