Python

Python é uma linguagem de programação de alto nível, lançada por Guido van Rossum em 1991. Atualmente, possui um modelo de desenvolvimento comunitário, aberto e gerenciado pela organização sem fins lucrativos Python Software Foundation.

Através do Curso Python Decodifica você será capaz de se tornar um desenvolvedor completo em Python, aprendendo os fundamentos básicos e intermediários da linguagem bem como Orientação a Objetos, Banco de Dados, Interface Gráfica, dentre outros. Aproveite!
